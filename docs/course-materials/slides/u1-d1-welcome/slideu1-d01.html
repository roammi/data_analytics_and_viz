Analítica de datos

¿Qué es analítica de datos?

El análisis es el proceso de descubrir, interpretar y transmitir patrones importantes presentes en los datos. De manera sencilla, el análisis nos permite ver estadísticas y datos importantes que de otra forma podríamos no detectar. 1

  1. Oracle - Data Analytics

¿Qué es analítica empresarial?

El análisis empresarial se dedica a utilizar las estadísticas derivadas de los datos para tomar decisiones más informadas que ayuden a las organizaciones a aumentar las ventas, reducir los costos y mejorar otras cuestiones empresariales.1

  1. Oracle - Data Analytics

Proceso de Ciencia de Datos

Fuente: R para Ciencia de Datos

Proceso de Ciencia de Datos

  1. Importar los datos. Típicamente, esto implica tomar datos que están guardados en un archivo, base de datos o API y cargarlos como data frame o extracto.
  2. Ordenar cuando tus datos están ordenados, cada columna es una variable y cada fila una observación, con estructura consistente.
  3. Transformarlos implica reducir las observaciones a aquellas que sean de interés, crear nuevas variables que sean funciones de variables ya existentes y calcular una serie de estadísticos de resumen.
  4. Visualización, te mostrará cosas que no esperabas o hará surgir nuevas preguntas acerca de los datos. También puede darte pistas acerca de si estás haciendo las preguntas equivocadas o si necesitas recolectar datos diferentes.
  5. Modelos, son herramientas complementarias a la visualización. Una vez que tus preguntas son lo suficientemente precisas, puedes utilizar un modelo para responderlas.
  6. Comunicación, una parte crítica de cualquier proyecto de análisis de datos. No importa qué tan bien tus modelos y visualizaciones te hayan permitido entender tus datos, a menos que también puedas comunicar esos resultados a otras personas.

Proceso de Exploración de Datos

La exploración de datos es el arte de mirar tus datos, generar hipótesis rápidamente, testearlas con celeridad y luego repetir el proceso de manera iterativa.

El objetivo de la exploración de datos es generar muchos hallazgos prometedores que luego puedas retomar para explorarlos en mayor profundidad. 1
